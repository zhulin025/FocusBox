# FocusBox 🦊

macOS 屏幕演示高亮工具 - 通过拖动鼠标绘制彩色渐变矩形框，让屏幕演示更清晰！

## ✨ 功能特性

- 🎨 **随机渐变边框** - 每次拖动生成独特的渐变色彩
- 🖱️ **简单易用** - 按住鼠标左键拖动即可绘制
- ⏱️ **自动消失** - 松开鼠标后 1 秒自动淡出
- 📱 **触摸板支持** - 完美支持 MacBook 触摸板
- 🎯 **全局监听** - 在任何应用上都能使用

## 🚀 使用方法

1. 运行 FocusBox 应用
2. 在任意位置**按住鼠标左键拖动**
3. 看到彩色矩形框后**松开鼠标**
4. 矩形框会在 1 秒后自动消失

## 📦 构建

```bash
./build.sh
```

构建完成后，应用位于 `build/FocusBox.app`

## ⚠️ 权限要求

首次运行需要授予**辅助功能权限**：

1. 打开 **系统设置** → **隐私与安全性** → **辅助功能**
2. 添加 FocusBox 并启用权限

## 🛠️ 技术栈

- Swift 5
- AppKit
- SwiftUI (设置界面)

## 📝 说明

- 状态栏图标：右上角虚线矩形图标
- 设置窗口：启动时自动显示
- 日志文件：`/tmp/focusbox.log`

## 🎨 示例

拖动鼠标时会看到：
- 彩色渐变边框（每次颜色不同）
- 半透明填充效果
- 平滑的淡出动画
