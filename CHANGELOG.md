# FocusBox v1.0.0 更新说明

## 📅 2026-03-11

### ✨ 新增功能

1. **淡出延迟调节** ⏱️
   - 可在设置窗口中调节矩形框停留时间
   - 范围：0.5 秒 - 5 秒，步进 0.5 秒
   - 默认值：1.0 秒

2. **屏幕录制功能** 🎬
   - 快捷键 `⌘+⇧+R` 开始/停止录制
   - 使用 AVFoundation 框架实现真正的视频录制
   - 录制文件保存至 ~/Downloads/ 目录
   - 格式：MP4，30 FPS，高清质量
   - 录制开始/完成时发送系统通知

3. **状态栏图标增强**
   - 启用状态：虚线矩形图标
   - 暂停状态：带 X 标记的图标 + ⏸️ 表情

### 🔧 优化改进

1. **代码质量**
   - 修复编译器警告
   - 改进日志输出
   - 优化内存管理

2. **用户体验**
   - 设置窗口尺寸调整（350x450）
   - 更清晰的使用说明
   - 完善的通知系统

### 📋 使用说明

#### 绘制矩形框
1. 按住鼠标左键拖动
2. 看到彩色渐变边框
3. 松开鼠标
4. 矩形框在设定延迟后自动淡出

#### 录制屏幕
1. 按 `⌘+⇧+R` 开始录制
2. 收到"录制已开始"通知
3. 再次按 `⌘+⇧+R` 停止录制
4. 收到"录制已保存"通知
5. 文件保存在 ~/Downloads/

#### 调节设置
1. 点击状态栏图标 → "显示设置窗口"
2. 或直接按 `⌘+S`
3. 调节边框粗细、颜色主题、淡出延迟

### ⚠️ 权限要求

**必须权限：**
- 辅助功能权限（绘制功能）
- 屏幕录制权限（录制功能）

**获取方式：**
系统设置 → 隐私与安全性 → 选择对应权限 → 添加 FocusBox

### 📊 技术指标

- **编译目标**：macOS 12.0+
- **架构**：x86_64
- **依赖框架**：AppKit, SwiftUI, AVFoundation, UserNotifications
- **应用大小**：~1.2 MB
- **内存占用**：< 50 MB
